DVD Central Project

DVD Central is a e-commerce website where customers can browse for and purchase movies. During my ASP.NET class, I created this website as part of the coursework. While it lacks aesthetics, I was able to learn the in's and out's of ASP.NET, Entity Framework Core, and Bootstrap CSS. I gained confidence in my back-end programming skills, along with using a remote database, throughout the course of this 17 week project. I was also introduced to Azure Devops, where I learned and practiced using project management software.
